# Documentation index

All guideline files for this template live in **`docs/`**. Read order for new contributors:

1. **`../README.md`** — clone, install, run, what to change first.
2. **`ARCHITECTURE.md`** — where code goes.
3. **`TECHNICAL.md`** — React ↔ Phaser ↔ store ↔ popups.
4. **`GAME_SPEC.md`** — fill in for *your* game (living doc).
5. **`DESIGN.md`** — Sunflower Land design philosophy.
6. **`UI_UX_GUIDELINES.md`** — panels, labels, mobile-first UI.
7. **`API.md`** — stub API surface.
8. **`ART.md`** / **`AUDIO.md`** — asset rules.
9. **`VALIDATION.md`** — manual pre-ship checklist.

**Agents:** each doc starts with an **Agent summary** and **MUST / MUST NOT** where applicable.
